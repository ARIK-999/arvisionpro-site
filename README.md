# AR Vision Pro

AR Vision Pro is a browser-based augmented-reality interface built with HTML, CSS, JavaScript, Tailwind CSS, MediaPipe, and TensorFlow.js.

Live site: https://arvisionpro.netlify.app/

## Project files

- `index.html` - app markup and external library loading.
- `styles.css` - custom glass UI, animation, canvas, and scrollbar styles.
- `tailwind.config.js` - Tailwind CDN configuration.
- `app.js` - camera startup, MediaPipe hands/pose tracking, TensorFlow object detection, drawing tools, language controls, and theme controls.
- `netlify.toml` - Netlify publish settings for the static site.

## Deploy

This project can be deployed as a static site from the repository root. Netlify publish directory: `.`.
