# AR Vision Pro

AR Vision Pro is a browser-based augmented-reality interface built as a single-page static web app.

Live site: https://arvisionpro.netlify.app/

## Project files

- `index.html` - the full app, including styles and scripts.
- `netlify.toml` - Netlify publish settings for the static site.

## Deploy

This project can be deployed as a static site from the repository root. Netlify publish directory: `.`.
